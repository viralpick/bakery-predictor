# 마감할인 실수요 검증 — Phase A 결과 (정직본)

**Date**: 2026-07-04
**매장/범위**: 광교(보나비), bread·pastry (sandwich=단일품목·cake=시즌 제외), 2021~2025
**질문**: 마감할인으로 팔린 물량 C 중 *진짜 수요*(제값에도 샀을 손님) B의 비율 **α = B/C**는 얼마인가? (모델링 결정 `실수요 = 정상판매 + 마감판매 × α`의 α. 기존 추측 0.5~0.7.)

## 요약 (한 줄)

**광교 데이터로는 α를 특정 숫자로 못 박는다.** 저녁 마감할인이 5년간 거의 매일 상시라 행동기반 식별이 confound에 막힌다. 단, 증거는 **"마감물량 상당수가 진짜 수요"** 쪽으로 기운다 — 저녁에 손님이 줄지 않고(발길 안정), 마감 물량이 잔량에 강하게 비례하지도 않는다(순수 떨이 아님).

## 방법 (3각 식별) 과 각 한계

| 방법 | 무엇을 재나 | 실측(bread / pastry) | 판정 |
|---|---|---|---|
| **A1** kink-in-time | 마감 onset 직전(오후) 판매율을 counterfactual로, 초과분=유도 | α_raw = 0.851 / 0.836 | ⚠️ floor 미검증 |
| **A2** depth elasticity | 20% vs 30% 할인 반응 → 0%로 외삽한 base | **degenerate (NaN)** | ❌ depth endogeneity |
| **A3** surplus counterfactual | 마감 물량이 실측 잔량(=마감+폐기)에 비례하나 | slope 0.286 / 0.383 (<0.5) | △ directional만 |
| **집계 alpha_low** | 유효 하한 방법의 max | **NaN (유효 하한 없음)** | — |

### A2가 degenerate인 이유
30%(0077) 물량이 20%(0069)의 약 4.7배 → 두 점을 0%로 선형 외삽하면 base가 음수 → 무의미. depth 선택이 내생적(잔량 많은 날 깊게 할인). depth_time_separated=False(둘 다 21시 중앙값)라 시간 confound와도 얽힘. → **A2는 이 데이터에서 정보 없음.**

### A1이 floor가 아닌 이유 (핵심 발견)
A1은 "저녁 수요 ≥ 오후 수요"일 때만 하한이다. 검증:
- 저녁(20-21시) **비할인** 판매율이 오후(17-19시)의 1/5~1/7 → 처음엔 "저녁 손님 감소, A1 과대"로 읽음.
- **그러나 총 발길(할인 무관)을 보니 반대**:

| | 오후 17-19 | 저녁 20-21 | 비율 |
|---|---|---|---|
| 총 영수증/시간 | 441 | 482 | **1.09** |
| 총 수량/시간 | 32,810 | 33,004 | **1.01** |
| 마감할인 비중 | ~5% | **91-93%** | — |

→ **저녁에 손님은 줄지 않는다(오히려 약간 증가). 다만 91%가 할인품을 산다.** 즉 낮은 비할인율은 수요 부재가 아니라 **미끼상품 잠식**(옆에 할인품 있으니 정가를 안 삼)의 결과. `traffic_stable=True, a1_bias=indeterminate`.
→ 그래서 A1이 α를 과대추정한다고 **단정할 수 없다**. 오히려 counterfactual로서 오후 rate가 타당할 수 있다.

### 그래도 α를 확정 못 하는 이유 (남은 confound)
**5년간 매일 저녁 할인이 손님의 방문 시간을 내생적으로 훈련**시켰다. "저녁 발길 안정"조차 할인이 만든 균형이다. 할인을 없앴을 때 저녁 손님이 (i) 오후로 옮겨 정가 구매(=실수요, 높은 α)할지 (ii) 안 살지(=유도, 낮은 α)를 발길만으론 못 가린다. → α 미확정.

## 정직한 결론

- **α의 점/구간 확정: 불가** (alpha_low=NaN — 검증된 하한 없음).
- 말할 수 있는 것:
  1. **손님은 저녁에 줄지 않는다** (footfall 안정) — "떨이라 억지로 온 손님뿐"은 아님.
  2. **순수 공급주도 떨이가 아니다** (A3 demand-limited, 마감 물량이 잔량에 약하게만 반응).
  3. → 증거는 **높은 α**(마감물량 상당수가 진짜 수요) 쪽으로 기울지만, 내생적 timing confound로 숫자 확정은 못 함.
- 실무 함의: `실수요 = 정상 + 마감×α`에서 α를 낮게(0.5) 잡으면 실수요를 과소평가할 위험. 다만 정확한 α는 Phase B/다중검증 대기.

- 광교 단독. 4매장 확장은 후속(parquet에 4매장 있음).
- 고객 ID 없음 → 딜헌터 개인 식별 불가, 행동(elasticity/footfall)에서만 간접.

- α 미확정은 데이터 한계이지 방법 실패가 아님 — 저녁 상시 할인이라는 광교 특성의 결과.
