# 분포 부스팅 특수일 PoC 결과

NGBoost(LogNormal, σ(x) 학습) vs 현 스택(LightGBM q0.5/q0.85).
walk-forward expanding 8×8주, target=adjusted_demand_unit, NGBoost n_est=500/lr=0.02.

설계: `docs/superpowers/specs/2026-07-20-distributional-boosting-special-day-poc-design.md`

## 판정: **분포회귀(NGBoost) 채택 정당 — 곱셈 구조만으론 부족**

광화문 corr: 선형 -0.55 / LightGBM-log -0.60 / NGBoost +0.17; 이벤트 폐기율(+prior) log 0.223 / NGBoost 0.163. ★판별 실험이 '그냥 log-변환하면 되지 않나'를 **반증**: log-공간 독립분위수도 병리가 그대로(-0.60, 선형 -0.55과 동급). 오직 **결합 분포 적합(NGBoost가 μ·σ를 proper scoring으로 동시추정)**만 병리를 반전. 원인=독립분위수는 log에서도 저수요(sparse) 구간 상단분위수가 불안정 → 마진이 되레 벌어짐. 결합적합은 σ(x)를 매끈한 함수로 학습해 이를 제거. 이벤트 폐기도 NGBoost 우위. ⇒ **분포회귀 전환 근거 확보**(단순 target 로그화로는 안 됨).

### 1) point-WAPE 패리티 게이트 (전제 — 통과)
전체 WAPE NGBoost 0.086 ≈ LightGBM 0.087 ≈ LightGBM-log 0.089. 엔진 품질 차이가 아니므로 분포/스프레드 효과를 접근법에 귀속할 수 있다.

### 2) spread 진단 (주 증거 — 통계적으로 안정)
판별 핵심 = 선형 독립분위수(현 스택) vs 곱셈(log·LogNormal). 마진-레벨 상관:
- 광화문: 선형 **-0.55** (병리: 저수요일 마진 26 > 고수요일 15) / LightGBM-log **-0.60** / NGBoost **+0.17**
- 광교: 선형 +0.18 / log +0.15 / NGBoost +0.55

> **메커니즘(데이터 검증)**: 핵심 구분 = '곱셈 공간(log target)' ≠ '곱셈 결합(공유 μ)'. LogNormal은 q_a=exp(μ+σ·z_a)라 두 분위수가 공유 μ에 묶여 절대마진이 **구조적으로** median에 비례. LightGBM-log는 q50·q85를 **독립 트리**로 fit → 선형갭이 레벨에 안 묶여 저수요일에 벌어짐. 광화문 tertile 검증(아래 표): 저수요일 NGBoost σ=0.077(고수요 0.064)로 σ는 오히려 **크지만**, 곱셈 결합 덕에 NGBoost 절대마진 17≈19로 유지. 반면 log/선형 마진은 저수요일 25/25 > 고수요일 17/17로 벌어짐(병리). 즉 이득은 '학습된 σ가 작아서'가 아니라 **공유-μ 결합**에서 온다. 메모리 '×K(레벨비례)' 직관의 정답형 = 손튜닝 K가 아닌 결합 분포 적합.

### 3) 이벤트일 프론티어 (N=14, 참고)
prior 없이는 곱셈만으론 이벤트 레벨을 못 잡음(cov NGBoost 0.57 / LightGBM 0.64, 둘 다 nominal 0.85 미달; rare event σ 표본부족—설계 예견).
prior 결합 이벤트 폐기율: NGBoost+prior 0.163 / LightGBM-log+prior 0.223 / LightGBM(선형)+prior 0.245 (매진율 0.07/0.00/0.00).
> **지배 아님, 유리한 trade**: NGBoost+prior는 폐기를 크게 줄이는 대신 매진율 0.07(14일 중 1일) vs 베이스라인 0.00. 헌장 KPI(폐기=1차, 매진=부차)상 옳은 방향이나 '지배'가 아니라 폐기↔매진 trade임을 명시.

### 4) ablation·calibration 노트
- σ(x)/곱셈구조와 prior는 **보완**: 곱셈=일반 저수요일 과대마진 해소, prior=rare-event 레벨앵커.
- 전체 cov@.85가 모델 공통 ~0.77로 nominal 0.85 미달 = 전역 under-calibration(본 작업 무관, 기존 finding). 이벤트일 cov 0.93~1.00은 prior가 레벨을 밀어올린 것이지 base q0.85가 calibrated된 게 아님 → **conformal 보정이 올바른 후속**.
